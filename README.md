# Shopping Wishlist Chatbot

A simple shopping wishlist chatbot built with Python using the Flask web framework. The bot allows users to manage their shopping wishlist through a conversational interface.

## Features

- Add items to your wishlist
- Remove items from your wishlist
- View the current wishlist
- Conversational UI for easy interaction

## Technologies Used

- Python
- Flask
- HTML
- CSS (Bootstrap)
